Data Sources

This dataset contains 7 tables in CSV format and 1 import data from Canada population website;

1. CSV File : 2018 Sales, 2019 Sales, 2020 Sales, Products, Locations, Customers, Sales People
2. Website : https://www.california-demographics.com/


Business Request & User Stories

As the participants of the Power BI Workshop, we have identified the need to analyze the sales performance of the chosen business. The aim is to empower workshop participants with practical insights into key metrics related to sales to enhance the understanding of business intelligence and analytics.

Data Cleansing and ETL

Following steps were done in Power BI to transform this table to be ready for analysis purposes:

1. Remove Median Income, Land Area, Water Area and Time Zone Columns in Locations Table.
2. Duplicate State columns and split the column by number of character of 4 to form a result of ‘California’ to ‘Cali’ and ‘fornia’.
3. Remove columns that contain data of ‘fornia’ and Uppercased the ‘CALI’ text for improved data quality.
4. Rename previous duplicated columns as ‘State Short’ give better business friendly names.
5. Merge the table Locations and Populations Data.
6. Remove previous Populations Data column to replace new populations data that import from web.
7. Rename the Populations Data columns as ‘Population’.
8. Create new conditional column for Population Density.
9. Replace errors from Population Density columns that contain ‘null’ as ‘No Data’.

Calculations

The following calculations were created in the Power BI reports using DAX (Data Analysis Expressions):

Cumulative Sales =
CALCULATE([Total Sales],
FILTER(ALLSELECTED(Dates),
Dates[Date]<=MAX(Dates[Date])))

Total Cost = SUMX(Sales, Sales[Quantity] * RELATED(Products[Cost]))

Total Sales = SUMX(Sales, [Total Quantity] * Sales[Price])

Total Profit = [Total Sales] - [Total Cost]

Total Quantity = sum(Sales[Quantity])

Total Transactions = COUNTROWS (Sales)

Sales LY = CALCULATE([Total Sales], SAMEPERIODLASTYEAR(Dates[Date]))
